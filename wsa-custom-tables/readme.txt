Create responsive html tables for your site using Beaver Builder Page Builder.

== Description ==

Create responsive tables for your site on base Beaver Builder Page Builder(https://www.wpbeaverbuilder.com/?fla=321).  You can change the style of tables and create beautiful pages organizing a specific information. 

[Click here](http://bb.webstudiolab.com/wsl-tables-module/?lang=en "WSA Tables Module") to see the demo of this plugin.

The plugin only works with limited number of themes as of now, here is the list of supported themes - 

1. Beaver Builder theme
2. Genesis Theme
3. Webstudiolab Theme

== Installation ==

Download this plugin, install and activate as a usual Wordpress Plugin.


1. Upload the plugin files to the `/wp-content/plugins/ws` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. Use the Settings->Page Builder-> WSA Modules -> WSA Tables to add on the page a table.
